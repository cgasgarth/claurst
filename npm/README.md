# claurst

Install the Claurst CLI from prebuilt GitHub release binaries.

## Install

```bash
npm install -g claurst
```

## What this package does

This npm package is a thin wrapper around the Rust `claurst` binary published in GitHub Releases for `cgasgarth/claurst`.

On install it downloads the correct release asset for your platform and architecture, then exposes the `claurst` command on your PATH.

## Supported platforms

- macOS x64 / arm64
- Linux x64 / arm64
- Windows x64
